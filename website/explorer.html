Explorer

Inspect the network in real time.

Point the explorer at a live CURS3D node HTTP API to monitor chain status, recent blocks, validator activity and account or transaction state.

The explorer is API-driven. If the node is offline or private, the dashboard will stay empty until a reachable endpoint is configured.
The configured node is not reachable right now. Make sure a CURS3D node is running and the HTTP API is exposed.

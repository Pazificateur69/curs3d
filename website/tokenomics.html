Tokenomics

Tokenomics should explain value coordination, not just token supply.Les tokenomics doivent expliquer la coordination de valeur, pas seulement l’offre du token.

The native token has to secure the network, align governance, support validator economics and reflect how value flows through the protocol.

Le token natif doit sécuriser le réseau, aligner la gouvernance, soutenir l’économie validateurs et refléter la manière dont la valeur circule dans le protocole.

Token roleRôle du token

Fees

The token pays for network usage and execution.

Le token paie l’usage réseau et l’exécution.

Staking

The token secures validator participation and protocol safety.

Le token sécurise la participation validateurs et la sûreté du protocole.

Governance

Gouvernance

The token aligns decision-making and network evolution.

Le token aligne la prise de décision et l’évolution du réseau.

Alignment

Alignement

The token links network activity, treasury design and ecosystem growth.

Le token relie activité réseau, design de trésorerie et croissance écosystème.

Allocation philosophyPhilosophie d’allocation

CURS3D should avoid chaotic token design. A credible framework must balance founding incentives, validator alignment, treasury resilience, ecosystem grants and long-term lock discipline.

CURS3D doit éviter les tokenomics chaotiques. Un cadre crédible doit équilibrer les incentives fondatrices, l’alignement validateurs, la résilience de trésorerie, les grants écosystème et une discipline de lock long terme.

Network usage

Usage réseau

Network activity should generate defensible economic relevance for the token rather than purely narrative demand.

L’activité réseau doit générer une pertinence économique défendable pour le token, pas seulement une demande narrative.

Validator incentives

Incentives validateurs

Validators must be paid enough to sustain network quality, but not through incoherent or short-term inflation policies.

Les validateurs doivent être suffisamment rémunérés pour maintenir la qualité réseau, mais pas via des politiques inflationnistes incohérentes ou court-termistes.

Treasury logicLogique de trésorerie

Treasury design is part of protocol survivability. If CURS3D aims to be long-horizon infrastructure, the treasury must be able to support audits, operations, ecosystem support and incident response over time.

Le design de trésorerie fait partie de la survivabilité du protocole. Si CURS3D vise une infrastructure pensée sur le long terme, la trésorerie doit pouvoir soutenir les audits, les opérations, le support écosystème et la réponse aux incidents dans la durée.

Important noteNote importante

This page describes a tokenomics framework, not a final public token sale plan.

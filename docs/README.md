# CURS3D Docs

Ce dossier regroupe la documentation stratégique, de marque et de crédibilité de `CURS3D`.

Objectif:
- disposer d'une base sérieuse pour présenter le projet à des investisseurs, partenaires, validateurs et projets tiers
- unifier la thèse, le positionnement, la narration produit et la préparation mainnet
- éviter les incohérences entre site, deck, docs techniques et communication publique

Structure:
- [`branding/`](./branding/README.md): image de marque, slogans, ton, messages, garde-fous
- [`foundation/`](./foundation/README.md): thèse, marché, produit, gouvernance, tokenomics, fundraising, mainnet readiness
- [`../whitepaper.md`](../whitepaper.md): whitepaper source en Markdown

Ordre de lecture recommandé:
1. [`foundation/00-executive-summary.md`](./foundation/00-executive-summary.md)
2. [`foundation/01-curs3d-thesis.md`](./foundation/01-curs3d-thesis.md)
3. [`branding/01-brand-core.md`](./branding/01-brand-core.md)
4. [`foundation/13-litepaper.md`](./foundation/13-litepaper.md)
5. [`foundation/14-investor-deck-outline.md`](./foundation/14-investor-deck-outline.md)
6. [`foundation/15-mainnet-readiness-plan.md`](./foundation/15-mainnet-readiness-plan.md)

Principe:

`CURS3D` doit être présenté comme une `Layer 1 post-quantique` pensée pour les actifs numériques conçus pour durer, et non comme une simple blockchain expérimentale.

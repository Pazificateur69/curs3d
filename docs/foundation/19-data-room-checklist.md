# Data Room Checklist

## Objectif

Préparer un socle documentaire crédible pour discussions investisseurs, partenaires stratégiques et due diligence initiale.

## Dossier minimum

- executive summary
- thesis note
- litepaper
- investor deck
- roadmap 12 mois
- tokenomics framework
- governance framework
- risk memo
- mainnet readiness plan

## Dossier technique

- repo source
- instructions de build
- instructions de test
- résultats de tests
- architecture summary
- security assumptions
- release process

## Dossier opérationnel

- structure équipe
- rôles
- priorités 12 mois
- budget cible
- besoins de recrutement

## Dossier légal

- structure prévue
- hypothèses de juridiction
- politique de token planning
- liste des sujets à valider avec avocat

## Ce qu'il faut éviter

- documents contradictoires
- versions non datées
- promesses plus fortes dans le deck que dans les docs
- lacunes sur les risques

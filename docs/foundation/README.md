# CURS3D Foundation Docs

Ce dossier rassemble les documents de fond nécessaires pour présenter `CURS3D` comme un projet crédible, cohérent et structuré.

Objectif:
- clarifier la thèse de fond du projet
- unifier la narration technique, produit, marché et investisseur
- documenter les risques et les exigences de crédibilité
- préparer une base exploitable pour deck, site, partenaires, audits et levée

# Fundraising Readiness

## Ce qu'un investisseur veut voir

- thèse claire
- équipe fondatrice crédible
- protocole fonctionnel
- documentation sérieuse
- roadmap réaliste
- premiers signaux d'adoption
- gestion lucide des risques

## Matériel minimum

- deck 10 slides
- one-pager
- executive summary
- memo produit
- tokenomics propres
- data room basique

## Data room minimale

- code repo
- docs protocolaires
- roadmap
- audit plan
- token draft
- cap table si existante
- structure légale prévue

## Red flags à éviter

- promesses excessives
- chiffres non sourcés
- confusion entre communauté et traction
- tokenomics floues
- aucune réponse aux risques

## Position recommandée

Présenter `CURS3D` comme:
- une thèse forte
- un protocole qui se structure
- une catégorie à construire

Pas comme:
- un champion déjà prouvé
- un standard déjà adopté

# Mainnet Readiness Plan

## Objectif

Transformer `CURS3D` d'un projet techniquement intéressant en protocole présentable pour un lancement mainnet crédible.

## Phase 1: Protocol Hardening

À obtenir:
- build et tests propres
- politique de releases claire
- cohérence entre code, docs et site
- monitoring et métriques
- procédures de sauvegarde et restauration

Livrables:
- changelog
- release checklist
- incident playbook
- configuration node stable

## Phase 2: Security Validation

À obtenir:
- audit externe
- revue cryptographique
- bug bounty
- revue des endpoints publics
- revue de la persistance et des reorgs

Livrables:
- audit report
- plan de remédiation
- publication d'un security policy

## Phase 3: Product Readiness

À obtenir:
- docs développeurs lisibles
- endpoint RPC/API stables
- explorer utilisable
- wallet path défini
- guide opérateur nœud

Livrables:
- quickstart
- operator brief
- FAQ développeur

## Phase 4: Ecosystem Readiness

À obtenir:
- 3 à 5 projets pilotes
- cas d'usage démontrables
- narratif partenaire clair
- process de support et onboarding

Livrables:
- partner one-pager
- integration checklist
- programme de grants simple

## Phase 5: Governance and Token Readiness

À obtenir:
- gouvernance documentée
- tokenomics cohérentes
- règles d'upgrade et d'urgence
- structure opérationnelle claire

Livrables:
- governance framework
- tokenomics note
- multisig policy

## Phase 6: Public Launch Readiness

À obtenir:
- site cohérent
- deck investisseur prêt
- message public discipliné
- press kit minimum
- calendrier de lancement

Livrables:
- launch plan
- press kit
- FAQ publique

## Exit Criteria Mainnet

Ne pas lancer tant que les points suivants ne sont pas vrais:
- protocole stable
- docs cohérentes
- audit externe lancé ou terminé
- opérabilité démontrée
- thèse répétable par quelqu'un d'autre que le fondateur
- premiers partenaires identifiés

## Règle simple

Le mainnet ne doit pas être un acte de foi.  
Il doit être la conséquence d'un protocole, d'une narration et d'une opérationnalité déjà crédibles.

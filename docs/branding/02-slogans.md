# Slogans

## Slogan principal recommandé

`The Quantum-Secure Layer 1 for Digital Assets That Need to Last.`

Version française:

`La Layer 1 post-quantique pour les actifs numériques conçus pour durer.`

Pourquoi ce choix:
- dit clairement ce que fait CURS3D
- met l'accent sur la durée, qui est la vraie différenciation marketing
- évite le piège du slogan trop abstrait

## Signature courte recommandée

`Built to Endure.`

Version française:

`Conçue pour durer.`

Usage:
- footer
- hero secondaire
- merch
- decks

## One-liner principal

`CURS3D is a post-quantum Layer 1 designed for long-duration digital assets, tokenized value, and trust-critical on-chain infrastructure.`

## Alternatives premium

- `Security for the Long Horizon.`
- `Future-Proof Value, On-Chain.`
- `Quantum-Secure by Design.`
- `Infrastructure for Durable Digital Value.`
- `Where Long-Term Assets Belong.`

## Alternatives plus institutionnelles

- `A Durable Trust Layer for the Post-Quantum Era.`
- `The Layer 1 for Long-Term On-Chain Security.`
- `Digital Asset Security for the Next Cryptographic Era.`

## Alternatives plus agressives

- `Most chains scale. CURS3D prepares.`
- `Before the migration, there is CURS3D.`
- `Don't issue long-term value on short-term assumptions.`

## Slogans à ne pas utiliser

- `The fastest blockchain`
- `The next Ethereum`
- `Quantum blockchain revolution`
- `Military-grade blockchain`
- `AI x Quantum x Web3`

Pourquoi:
- trop faibles
- trop vagues
- trop marketing
- pas défendables
